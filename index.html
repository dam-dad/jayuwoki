Documentation

JavaFX

Libreria GUI para java

JavaFX nos permite crear la interfaz visual con la que interactua el usuario. Con ella somos capaces de crear ventanas y botones, entre otros elementos.

JDA y Spring

Librerias para crear el bot de discord

JDA es una libreria que nos permite interactuar con la API de discord, mientras que Spring nos permite crear un servidor web para la interacción con el bot.

Firebase

Base de datos

Firebase es una base de datos en tiempo real que nos permite almacenar la información de los usuarios y sus partidas.
